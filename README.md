# Bar-Decode

Clojure lirary for EAN-13 barcodes. Provides functions for encoding, decoding and validating.

## Usage

To scan an ASCII format PGM image for an EAN-13 barcode:

    $ java -jar bar-decode-0.1.0-standalone.jar <image.pgm>
